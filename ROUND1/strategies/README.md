# Strategies 

Backing up strategies `output.log` files to plug into [Jasper's visualizer](https://jmerle.github.io/imc-prosperity-3-visualizer/).

Follow instructions from the website's homepage to visualize the `output.log` file.

You can generate an `output.log` file using [prosperity4bt](https://pypi.org/project/prosperity4bt/), forked from `jmerle`'s own Prosperity 3 backtester [here](https://github.com/jmerle/imc-prosperity-3-backtester).
